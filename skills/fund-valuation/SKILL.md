# 基金实时估值

根据基金最新季报持仓和股票实时行情，估算基金当日净值变化。

## 使用方法

运行估值脚本，输入基金代码或名称：

```bash
# 使用基金代码
python scripts/valuation.py 007491

# 使用基金名称（支持模糊搜索）
python scripts/valuation.py 南方信息创新
python scripts/valuation.py 易方达蓝筹
```

## 输出内容

脚本会显示：

1. **基金基本信息**：名称、代码
2. **前10大重仓股明细**：
   - 股票代码和名称
   - 持仓占净值比例
   - 今日涨跌幅
   - 对基金净值的贡献
3. **估算结果**：
   - 前N大重仓股合计占比
   - 估算今日涨跌幅

## 估值原理

**计算公式**：

```
估算涨跌幅 = Σ(持仓占比 × 股票涨跌幅)
```

**示例**：
- 某股票持仓占比 10%，今日上涨 5%
- 贡献估值 = 10% × 5% = 0.5%

**注意事项**：

1. **数据来源**：
   - 持仓数据来自最新季报（通常滞后1-3个月）
   - 股票行情为实时数据
   
2. **估值局限**：
   - 仅计算前10大重仓股（通常占比70-90%）
   - 未包含其他持仓股票的涨跌
   - 未考虑基金经理的仓位调整
   - 未考虑申购赎回影响

3. **适用场景**：
   - 盘中快速了解基金大致表现
   - 判断重仓股对基金的影响
   - 参考性估值，非精确计算

## 数据源

- 基金持仓：天天基金网（eastmoney.com）
- 股票行情：东方财富实时行情API
- 更新频率：持仓数据每季度更新，股票行情实时

## 常见问题

**Q: 估值准确吗？**  
A: 仅供参考。实际净值受所有持仓影响，且基金经理可能已调仓。

**Q: 为什么只有前10大持仓？**  
A: 季报只公开前10大重仓股明细，其他持仓不公开。

**Q: 多久更新一次持仓？**  
A: 基金每季度末公布持仓，通常滞后1-2个月。

**Q: 支持哪些基金？**  
A: 支持所有在天天基金网有数据的公募基金。
